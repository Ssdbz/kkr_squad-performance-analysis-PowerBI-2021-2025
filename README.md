# KKR Performance Analysis (2021–2025) 

## Overview
This project analyzes the performance of Kolkata Knight Riders (KKR) over the last five IPL seasons (2021–2025) using Power BI.

The goal is to extract insights on player performance, team strategy, and match-phase contributions.

---

##  Key Insights
- Rahul Tripathi has been the highest run-scorer but remains underutilized
- Varun Chakravarthy’s dip in form has impacted bowling performance
- Rahane vs Narine highlights consistency vs explosiveness trade-off
- Rinku Singh emerges as the most reliable finisher
- Role clarity and player utilization remain key challenges for KKR

---

##  Tools Used
- Power BI
